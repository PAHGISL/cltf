# SA Minnipa Heavy/Imazapic shared inputs

The normalized observations, climate forcing, and soil inputs for this case
are consumed by `examples/R/run_reference_case.R` and both language
implementations. Expected outputs are stored under
`reference/sa_minnipa_heavy_imazapic/`.

This secondary regression case uses:

- 0–100 mm as the top layer and 100–300 mm as the bottom layer;
- application date `2024-06-12`;
- observed forcing through `2024-10-28`;
- an explicitly labelled provisional SLGA v2-shaped bulk-density fixture;
- concentration unit recorded as provisionally inferred `ug/kg dry soil`.
